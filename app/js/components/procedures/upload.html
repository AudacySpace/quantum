Download Template

  • File Type should be .xlsx only.
  • Step,Type,Role,Content fields are mandatory.Reference and Procedures columns are optional.
  • Only one type can be given for a step.Valid Step types include Heading, Action, Caution, Decision, Info, Record, Verify and Warning.
  • Role can be one or many(delimited by , if many).Valid roles are MD,CC,SYS,GC,PROXY,NAV,IT and SIM.
  • Sheet name of the file should always be Sheet1.
  • Make sure Step column has cell formatting set to text.Note:This will avoid rounding the step numbers when uploaded.
  • Numbering for Steps are in heirarchical order starting with x.0 as top sections.
  • Only Heading type step number should end with .0 . For example 1.0,1.2.1.0
  • Heading type steps are either section headers or subsection headers.That means they do not have checkbox when executing the procedure.
  • Last Step should not be of type Heading.
  • Reference and Procedures columns are optional.Reference field is used to provide valid URL to additional information.Procedures field is used to provide procedure index numbers described in the content of that step.More details in the template.
